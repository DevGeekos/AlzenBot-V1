# Base bot multifonction

Cette base vous aide à débuter dans le développement de votre bot !
Liste des fonctions :
- 👋 Bienvenue & Au revoir
- 🔨 Modération
- 🎁 Giveaways
- 🎮 Jeux
- ℹ️ Informations

**Pour le déployer, rien de plus simple**
1. Mettez un token dans la dernière ligne du fichier main.py
2. Lancez start.bat
3. Et voila !
